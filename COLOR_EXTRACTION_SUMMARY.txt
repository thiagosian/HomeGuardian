COLOR CONVERSION METHOD
================================================================================

All colors converted using HSL color model:
  H (Hue):        0-360 degrees
  S (Saturation): 0-100%
  L (Lightness):  0-100%

Conversion Formula Used:
  1. Convert HEX â†’ RGB (0-1 range)
  2. Calculate max/min RGB values
  3. Compute L = (max + min) / 2
  4. Compute S based on lightness
  5. Compute H based on dominant channel
  6. Format as "H S% L%" (no units)

Precision: All conversions rounded to nearest whole number
