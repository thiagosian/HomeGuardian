# HomeGuardian UI

Icon-based version control integration for Home Assistant.

## Features

✅ **Automatic Icon Injection** - History icons appear automatically in the native HA UI
✅ **Quick History Access** - View last 5 commits with one click
✅ **One-Click Rollback** - Restore previous versions safely
✅ **Native Integration** - No custom cards needed
✅ **Safety Backups** - Automatic backups before rollbacks

## Supported Entities

- ✅ Automations
- ✅ Scripts
- ✅ Scenes
- ✅ Blueprints
- ✅ Voice Assistants
- ✅ Dashboards

## Requirements

⚠️ **HomeGuardian Add-on** must be installed first!

Install the add-on from: https://github.com/thiagosian/HomeGuardian

## Installation

1. Install via HACS
2. Restart Home Assistant
3. Icons will appear automatically in the UI

No configuration needed!

## Usage

1. **Click history icon** next to any entity name
2. **View commit history** in the popup
3. **Click "Restore"** to rollback to any version
